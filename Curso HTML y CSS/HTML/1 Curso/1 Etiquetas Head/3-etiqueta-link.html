Atributo rel Descripción Ejemplo
Información Visual
stylesheet Establece una relación con un documento .css para aplicar sus estilos. <link rel="stylesheet" href="estilos.css" type="text/css">
icon Indica un icono para utilizar a modo de favicon (icono de pestaña). <link rel="icon" href="favicon.ico" type="image/x-icon">
Documento
alternate Indica una versión alternativa del documento actual. <link rel="alternate" href="otra_version.html" type="text/html" title="Otra Versión">
pingback Indica un enlace al que se debe hacer ping cuando se haga referencia a este documento. <link rel="pingback" href="pingback.php">
Posicionamiento SEO
canonical Establece el enlace real para el documento actual para evitar URL duplicadas. <link rel="canonical" href="url_canonica.html">
prev Indica si existe una URL anterior al documento actual. Útil en enlaces con varias partes. <link rel="prev" href="url_anterior.html">
next Indica si existe una URL siguiente al documento actual. Útil en enlaces con varias partes. <link rel="next" href="url_siguiente.html">
Información o Datos Extra
author Indica un enlace con más información sobre el autor del documento actual. <link rel="author" href="info_autor.html">
help Indica un enlace con información de ayuda. <link rel="help" href="ayuda.html">
search Indica un enlace donde se puede encontrar un buscador del sitio actual. <link rel="search" href="buscador.html">
license Indica un enlace con la información de licencia de la página actual. <link rel="license" href="licencia.html">
manifest Indica un enlace con una URL con un archivo .json con un manifiesto de aplicación. <link rel="manifest" href="manifest.json">
Atributos posicionamiento SEO (Google)
Atributo rel Atributo href Significado
canonical Dirección URL URL exacta que indica a Google la URL real del documento. Cualquier variación será descartada.
prev Dirección URL Si el documento actual tiene varias partes, indica la URL de la página anterior.
next Dirección URL Si el documento actual tiene varias partes, indica la URL de la página siguiente.
